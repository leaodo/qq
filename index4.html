O próximo passo é procurar um dispositivo para boot. Qualquer dispositivo, seja um sistema operacional instalado em um SSD de última geração ou um Live-DVD Linux para a instalação. O que depende, naturalmente, qual deles está programado para inicializar na BIOS/UEFI. Se você tem um sistema operacional instalado, ela procurará o bootloader no disco primário.
Dispositivo boot
O bootloader carrega o sistema operacional. Ou mesmo uma janela para que você escolha entre dois ou mais sistemas operacionais, como muitos usuários das distros Linux estão acostumados.
Dependendo do sistema, esse bootloader pode ser o GRUB ou LILO, eles armazenam as instruções para
carregar o sistema operacional respectivo, inicializando o carregamento completo até a tela de login.
